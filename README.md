# Temple Management System

## Project Overview

Temple Management System is a PostgreSQL-based database project designed to manage temple operations such as visitor records, donations, donation receipts, staff management, temple events, darshan records, VIP donors, and prasad distribution.

## Technologies Used

- PostgreSQL
- SQL
- Git
- GitHub

## Features

- Visitor Management
- Donation Tracking
- Donation Receipt Management
- Staff Management
- Temple Event Management
- Darshan Record Management
- VIP Donor Tracking
- Prasad Distribution Management

## Database Tables

3. donation_receipts
